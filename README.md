Basic Calculator Website: This is a simple calculator website built using HTML, CSS and JavaScript. It performs basic arithmetic operations like addition, subtraction, multiplication and division in a clean, easy-to-use interface.

Features
1. Addition, subtraction, multiplication, and division  
2. Clear display screen  
3. Button-based input (like a real calculator)  
4. Instant calculation results  
5. Simple and responsive design  

Technologies Used
1. HTML – Structure of calculator layout
2. CSS – Styling, button design and layout
3. JavaScript – Calculation logic and interaction  

Learning Outcomes
1. Handling button click events in JavaScript  
2. Writing basic calculation logic  
3. Updating the display dynamically  
4. Improving UI layout with CSS  

This project helped me strengthen my front-end basics and improve confidence in JavaScript.
